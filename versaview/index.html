VersaView 6300B Box PCss

Los Box PC Allen-Bradley® VersaView® 6300B están disponibles en varias formas factores y procesadores para adaptarse a su aplicación. Unidades de montaje de libros tienen un diseño sin ventilador que los hace óptimos para aplicaciones que requieren bajos mantenimiento. Las unidades más compactas de esta familia cuentan con Apollo Lake Procesadores Atom, lo que los hace ideales para la interfaz hombre-máquina (HMI). Aplicaciones de registro de datos y puerta de enlace IIoT. Las unidades de clase Atom también son disponibles como clientes ligeros preparados para ThinManager® y no están equipados con un sistema operativo o unidad de estado sólido (SSD).

Clientes ligeros VersaView 6300T Box

Los VersaView® 6300TBox cuentan con procesadores Intel Apollo Lake Atom, lo que los hace ideales para aplicaciones de interfaz hombre-máquina (HMI), puerta de enlace IIoT y registro de datos. Las unidades de clase Atom también están disponibles como clientes ligeros preparados para ThinManager® para arrancar de forma nativa en el software ThinManager sin una unidad de estado sólido (SSD) a bordo. Estas unidades tienen un diseño sin ventilador que las hace óptimas para aplicaciones que requieren poco mantenimiento, así como múltiples métodos de montaje para adaptarse a las necesidades de la aplicación.

VersaView 6300P Panel PC

Los Panel PC VersaView 6300P cuentan con Core i3, i5 e i7 de séptima generación de la plataforma Intel Kaby Lake H, lo que le permite manejar aplicaciones industriales complejas. Las tecnologías de pantalla táctil, que mejoran la facilidad de uso, incluyen resistiva analógica y capacitiva proyectada (para operación multitáctil). Este Panel PC está disponible en una variedad de tamaños y resoluciones de pantalla. Hay opciones de unidades de bisel estándar o de perl bajo. Ambos tienen resoluciones de hasta alta denición completa.
